### Installation

Requirements:
- The Linux (or macOS) distribution (**Debian 11** was used for development)
- Globally installed composer

Steps:

1.) To install all dependencies run:
```shell script
make install
```

2.) To check code standards run:
```shell script
make lint
```

3.) Add permission for execution:

```shell script
chmod -R +x ./bin/
```
